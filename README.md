# TicTacToe Game in Python

This is a simple two-player TicTacToe (Noughts & Crosses) game in Python

Currently this is a very basic version of the game

## Requirements
* numpy 1.24.2
* pygame 2.5.0

## How to run

Inside directory:

Initial setup:
``` bash 
pip install -r requirements.txt
```


To run,
```shell
python tictactoe.py
```
or 
```shell
python3 tictactoe.py
```
